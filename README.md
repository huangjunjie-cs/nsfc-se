# ✨ NSFC-SE

这个项目基本上源于 [fylimas/nsfc](https://github.com/fylimas/nsfc), 但是默认的许多配置和原版干净整洁相差较远，例如颜色的选择。
因此，在实际进行撰写时，我进行了一些调整和修改，力求尽可能的简洁朴素。

如果你想要原版的效果，请访问其 [repository](https://github.com/fylimas/nsfc)
